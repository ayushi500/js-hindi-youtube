/*Selector Meaning Example 'li:first-child' Selects the first
  • Apple 'li:last-child' Selects the last
  • Cherry 'li:nth-child(3)' Selects the third
  • Cherry 'li:nth-child(odd)' Selects all odd list items 1st, 3rd, 5th 'li:nth-child(even)'Selects all even list items 2nd, 4th ⚠️ Important Tip querySelector() only returns the first matching element. If you want all matching elements, use: document.querySelectorAll('li:nth-child(2)'); → It returns a NodeList of all matches (you can loop through it). */
